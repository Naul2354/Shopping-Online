# Welcome Shopping Online Web app for purpose study 👋
<br>

## 🐧🐧cloning project from this link 

```
git clone https://github.com/Naul2354/nodejs-todo-app-demo.git
```

## 💡 install dependencies before run app

```
yarn install
```

<p>or</p>

```
npm install
```

## 🔎 run server

```
yarn run dev
```

<p>or</p>

```
npm run dev
```
## 🔎 run client-admin

```
yarn start
```

<p>or</p>

```
npm start
```
